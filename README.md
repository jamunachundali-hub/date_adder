# 📅 Date Adder 📂

![Date Adder Icon](file:///C:/Users/Asus/.gemini/antigravity/brain/b64bf4d4-0a5f-497b-a40d-de6fadf7f0b7/date_adder_icon_1773841325888.png)

A sleek, automated tool that helps you stay organized by **automatically prepending today's date** (`YYYY-MM-DD`) to every file name in a specified directory. Perfect for daily reports, document tracking, and digital asset management.

### 🌟 Key Features
- **Effortless Organization**: Instantly adds a clear time-stamp to all files.
- **Smart Safeguards**: Skip directories and only rename files if they haven't been dated yet.
- **Minimalist Interface**: Simple CLI tool that works with a single command.

---

### 🚀 How to Run

Running the script is straightforward. Follow these steps:

1. **Open your terminal** or command prompt in the project folder.
2. **Execute the script** using Python by specifying the folder you want to process.

#### Run in Current Directory:
```bash
python date_adder.py
```

#### Run for a Specific Folder:
```bash
python date_adder.py "C:\path\to\your\folder"
```
